# Homebrew Tap

Personal Homebrew tap for various tools.

## Installation

```bash
# Add the tap
brew tap jiikko/tap

# Install tt-client
brew install jiikko/tap/tt-client
```

## Available Formulae

### tt-client

ThumbnailThumb API client - Command-line interface for ThumbnailThumb app.

**Requirements:**
- ThumbnailThumb app must be running ([ThumbnailThumb](https://jiikko.com/thumbnail-thumb/))

**Usage:**
```bash
tt-client /help
```

## Development

To install from HEAD (develop branch):

```bash
brew install --HEAD jiikko/tap/tt-client
```
